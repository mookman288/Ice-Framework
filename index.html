Ice Framework

By PxO Ink

The Ice Framework is a lightweight modifiable and reusable system that can provide a basic structure for implementing fluid and accessible CSS designs. This page represents examples of what you can do with this framework.

By default, this framework expects all JavaScript to run after the page is loaded. You may choose to leverage certain JavaScript when the page is ready, rather than when the page is loaded.
